# Expense and Income Tracker App

A simple and interactive budgeting tool built with **HTML, CSS, and JavaScript**.  
Users can add, delete, and manage income and expense entries while keeping track of their total balance in real time.  

---

## Screenshot

![Expense Tracker Screenshot](screenshot.png)

---

## Features

- Add multiple income and expense entries with description, type, amount, and date  
- Delete any entry dynamically  
- Automatic calculation of total balance (income minus expenses)  
- Real-time updates whenever a new entry is added or removed  
- Responsive and user-friendly interface  

---

## How to Run
1. Clone or download the repository  
2. Open `index.html` in your web browser  
3. Start adding incomes and expenses to track your budget
